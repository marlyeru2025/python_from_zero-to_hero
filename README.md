# python_from_zero-to_hero
 Python Written Assignment
This repository contains a Jupyter Notebook with a set of Python exercises designed to practice core programming concepts such as lists, functions, loops, conditions, and user input.
The notebook includes 10 questions, each focusing on a different Python skill.

 Topics Covered
- Working with lists (add, remove, sort, modify)
- Writing and using functions
- Taking input from the user
- Conditional statements (if / elif / else)
- Loops (for & while)
- Simple games and logic (Number Guessing Game)
- Basic data analysis with lists
